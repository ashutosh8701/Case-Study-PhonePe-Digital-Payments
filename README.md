PhonePe Digital Payments Analytics Case Study
Overview

This project analyzes PhonePe digital payment transaction data across Indian states and districts to uncover patterns in transaction behavior, user adoption, device ecosystem trends, and demographic influences on digital payment usage.
Using Python, Pandas, Matplotlib, and Seaborn, the analysis combines transaction, user, device, and demographic datasets to generate business insights and actionable recommendations for market expansion and user growth.

•Analyzed 20.5B+ digital payment transactions across 36 Indian states using Python and Pandas, identifying that the
top 5 states contributing 55.16% of total transaction volume, revealing significant market concentration and expansion
opportunities.
• Conducted digital payment adoption analysis across 723 districts by integrating district-level transaction and demographics
datasets, uncovering a 0.31 moderate positive correlation between population density and transaction volume, suggesting
that denser districts tend to exhibit higher digital payment activity.
• Analyzed smartphone ecosystem trends across PhonePe users by evaluating state-level device usage patterns, identifying
Xiaomi with a 45.53% user share in Manipur, driving regional acquisition and partnership strategy.
• Developed state-level monetization metrics using transaction value and registered user data, identifying Telangana as the
highest-value market with Rs 30.7K average transaction amount per user, followed by Karnataka and Andhra Pradesh.
